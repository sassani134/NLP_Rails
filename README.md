# Rails NLP

## Instalation
### Prérequis : Avoir un environement Ruby, Rails
- Ruby:ruby 3.2.2
- Rails 7.0.8
###
Mettre la clé D'api dans blog_controller js
### Commande
 ``` $ cd NLP_Rails```
 ``` $ bundle install ```
 ``` $ bin/dev ```

Aller sur le site http://127.0.0.1:3000/
écrire du texte
fetch

et Voila
